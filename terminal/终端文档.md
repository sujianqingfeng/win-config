# Oh My Posh

> [oh-my-posh 下载](https://ohmyposh.dev/)

```
Install-Module oh-my-posh -Scope CurrentUser
```

ps1 文件放到 C:\Users\SmartSpace\Documents\WindowsPowerShell

```
// 查看配置文件
echo $profile

// 可编辑配置文件
nvim $profile.CurrentUserCurrentHost
```
